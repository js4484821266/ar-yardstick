# AR Yardstick

A Copilot-built AR yardstick app for Android that uses ARCore to measure real-world distances through your device's camera.

## Key Features

- **Arbitrary Mode**: Tap to place points and automatically measure distances between them
- **Standard Mode**: Use a 30cm reference line to estimate measurements
- **High Contrast Display**: 12-point measurement text with white-on-black for easy reading
- **Quick Exit**: Navigation bar always visible for fast app closing
- **Simple Controls**: Switch modes, clear measurements, and intuitive tap-to-measure interface

## Quick Start

1. Open the app on an ARCore-supported Android device
2. Grant camera permission
3. Wait for surface detection
4. Tap surfaces to measure (Arbitrary mode) or use the reference line (Standard mode)
5. Switch modes anytime with the "Switch Mode" button

See [BUILDING.md](BUILDING.md) for detailed build instructions and technical documentation.

## Requirements

- Android 7.0+ (API 24+)
- ARCore-supported device
- Camera permission
